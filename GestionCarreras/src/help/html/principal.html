Ventana principal

Pulsando F1 en la ventana principal, se abrira esta pagina de ayuda.

La ventana tiene un boton para abrir la gestion de corredores, que tambien tiene su propia pagina de ayuda.

Ademḿas tiene otro boton para abrir la gestion de carreras, con su pagina de ayuda también.

En la barra de menú se disponen también de varias opciones:

  • Archivos
  • Configuración
  • Ayuda

Cada menú desplegable tiene sus propias opciones y atajos de teclado que se detallan en en cada página de ayuda.

Tanto si se cierra la ventana pulsando la x como si se pulsa el boton Salir, se guardan los datos de la aplicación y se finaliza la ejecución de la misma.
